# 🧱 1️⃣ Core Security Principles (The Bedrock)

Understand these deeply:

## 🔒 Confidentiality
Only authorized users can access data.

## 🛡 Integrity
Data cannot be altered without authorization.

## ⚡ Availability
Systems and data must be accessible when needed.

## 🔑 Authentication
Verifying identity.

## 🧾 Non-repudiation
Users cannot deny actions they performed.

These form the **CIA + A + N model**.

Everything in cybersecurity connects back to these.

# 🔐 4️⃣ Cryptography (Data Protection Core)

Understand the difference between:

## 🔁 Symmetric Encryption
- Same key for encrypt/decrypt
- Fast
- Example: AES

## 🔐 Asymmetric Encryption
- Public & private keys
- Used for key exchange
- Example: RSA

## 🖊 Digital Signatures
- Verify integrity & authenticity

## 🏛 Certificate Authorities (CAs)
- Issue trusted certificates
- Enable HTTPS

Encryption protects confidentiality & integrity.
